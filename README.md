## ðŸš¦ How to Use

Polychronos is designed to be the "System Pilot" for an LLM-based IDE.

1.  **Load the System Prompt**: Feed `PolyChronos-Omega.md` into the AI's system context.
2.  **Initialize the Project**: The AI will start with the **Project Manager** and **Blueprint** phase.
3.  **Route the Task**: The **Router** determines if the task is a simple fix (T1) or a full build (T2/T3).
4.  **Execute via Agents**: The AI switches personas (e.g., "Acting as **Savant Architect**") to perform the work according to the agent contracts in `polychronos/agents/`.
